# helcli-example
An example CLI using HelCLI.

### Install
```
pip install git+https://github.com/sunshinekitty/helcli-example.git
```

### Usage
```
$ helcli -h
usage: helcli [-h] {test} ...

An example CLI

positional arguments:
  {test}
    test      Prints success for testing

optional arguments:
  -h, --help  show this help message and exit
```
